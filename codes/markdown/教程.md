# 特殊符号打法

| 符号             | 名称   | 语法         | 注意 |
| ---------------- | ------ | ------------ | ---- |
| $\widehat{a}$  | 预测值 | \widehat{a}  |      |
| $\sum_{i=0}^n$ | 求和   | \sum_{i=0}^n |      |
| $a_i$          | 下标   | a_i          |      |
| $a^2$          | 上标   | a^2          |      |
| $\frac{a}{b}$  | 分数   | \frac{a}{b}  |      |

# 插入视频

```markdown
<video controls="" preload="none"><source src="./test.mp4" type="video/mp4"></videos>
```

`preload="none"` 表示不自动播放，可以达到省流的目的
