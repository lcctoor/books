# 安装Python

## 下载软件包

[https://www.python.org/downloads/](https://www.python.org/downloads/)

## 安装软件

在开始安装前，须先注意以下几点：

1. 选择 `Install Now` 而不是 `Customize installation` 。
2. 安装过程中，所有可选框全部打勾。
3. 如果有看到 `Disable path length limit` ，建议点击该选项。

## 校验安装

在命令行分别输入以下指令，如果都有响应，则表示安装成功：

* `python --version`
* `pip`
